Блок навигации
  • Ширина 15em
  • Должен присутсвовать цвет фона
  • Расоплогаеться слева
  • Блок навигации наполните контеном достаточным, чтобы по высоте захватыввать вторую стаью, как показано в данном приемре

Статья 1

Размер блока:

Занимает всю доступную ширину

Должен присутсвовать цвет фона

Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom

Статья 2

Размер блока:

Занимает всю доступную ширину

Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
